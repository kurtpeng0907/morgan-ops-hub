# 管理營運系統

內部營運管理中樞單頁網站，包含預約派班、顧客資料、預約資料管理、人事權限、班表、財務報表與按摩師個人中樞。

## 使用方式

直接開啟 `index.html`，或用本機伺服器啟動：

```bash
python3 -m http.server 4173
```

開啟：

```text
http://localhost:4173/
```

## 檔案

- `index.html`：主要頁面
- `styles.css`：介面樣式
- `app.js`：系統功能邏輯
- `screenshot-overview.png`：總覽截圖
- `screenshot-appointment.png`：預約截圖
